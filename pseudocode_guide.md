# 邏輯競賽前哨站：虛擬碼 (Pseudocode) 與符號速查表

這份指南是為了幫助你在比賽前熟悉「虛擬碼」與常見的「邏輯符號」。
虛擬碼不是真正的電腦程式，而是一種**用人類語言寫給電腦看的邏輯草稿**。

---

## 1. 常見符號說明

### 🔢 算術符號 (數學運算)
| 符號 | 意義 | 範例 | 說明 |
| :---: | :--- | :--- | :--- |
| `+` | 加法 | `A + B` | A 加 B |
| `-` | 減法 | `A - B` | A 減 B |
| `*` | 乘法 | `A * B` | A 乘以 B (注意電腦用星號，不是 x) |
| `/` | 除法 | `A / B` | A 除以 B |
| `%` | 取餘數 (Mod) | `7 % 3` | 結果是 `1` (因為 7 除以 3 商 2 餘 **1**) |

### ⚖️ 比較符號 (比大小)
| 符號 | 意義 | 範例 | 結果 |
| :---: | :--- | :--- | :--- |
| `>` | 大於 | `5 > 3` | ✅ 真 (True) |
| `<` | 小於 | `2 < 8` | ✅ 真 (True) |
| `>=` | 大於或等於 | `A >= 10` | 若 A 是 10, 11, 12... 為真 |
| `<=` | 小於或等於 | `B <= 5` | 若 B 是 5, 4, 3... 為真 |
| `==` | 等於 | `A == B` | 判斷 A 和 B 是否相等 (注意是兩個等號) |
| `!=` | 不等於 | `A != B` | 判斷 A 和 B 是否**不**相等 |

### 📥 賦值符號 (設定數值)
| 符號 | 意義 | 範例 | 說明 |
| :---: | :--- | :--- | :--- |
| `←` 或 `=` | 設定/賦值 | `Score ← 100` | 把 `Score` 這個變數設為 100 |
| | | `Count = Count + 1` | 把 `Count` 目前的值加 1 後，再存回去 |

### 🧠 邏輯符號 (判斷條件)
| 關鍵字 | 符號 | 意義 | 範例 | 說明 |
| :---: | :---: | :--- | :--- | :--- |
| **AND** | `&&` | 且 (同時成立) | `A > 5 AND B < 3` | A 要大於 5 **而且** B 要小於 3，整句才算對 |
| **OR** | `||` | 或 (只要一個成立) | `A > 5 OR B < 3` | A 大於 5 **或者** B 小於 3，只要有一個對，整句就算對 |
| **NOT** | `!` | 非 (相反) | `NOT (A > 5)` | 如果 A 大於 5 是對的，加上 NOT 就變成錯的 |

## 2. 基本邏輯結構

### 🔹 條件判斷 (IF...THEN)
用來決定要不要做某件事。

```text
IF (今天下雨) THEN
    帶雨傘
ELSE
    戴帽子
END IF
```
*翻譯：如果今天下雨，就帶雨傘；否則（沒下雨），就戴帽子。*

### 🔹 迴圈 (Loop)
用來重複執行某個動作。

**1. FOR 迴圈 (固定次數)**
```text
FOR i FROM 1 TO 5 DO
    拍手
END FOR
```
*翻譯：從 1 數到 5，每數一次就拍手一次 (共拍 5 次)。*

**2. WHILE 迴圈 (滿足條件就一直做)**
```text
WHILE (肚子餓) DO
    吃餅乾
END WHILE
```
*翻譯：只要覺得肚子餓，就繼續吃餅乾 (直到不餓為止)。*

## 3. 實戰範例

**題目：** 計算 1 到 10 之間所有偶數的總和。

**虛擬碼寫法：**
```text
Sum ← 0                 // 設定總和一開始是 0
FOR i FROM 1 TO 10 DO   // 讓 i 從 1 跑到 10
    IF (i % 2 == 0) THEN  // 如果 i 除以 2 的餘數是 0 (表示是偶數)
        Sum ← Sum + i   // 把 i 加到總和裡
    END IF
END FOR
PRINT Sum               // 顯示最後的總和
```
